# Design Guidelines — minimal single-page landing

## Overall
Modern, minimal, uncluttered. One centered hero section that fills the viewport. Generous whitespace. Mobile-first, fully responsive.

## Color
- Background: clean white (#FFFFFF) / very light neutral.
- Text: near-black (#111827) headline, muted gray (#6B7280) subtext.
- One accent color for the CTA button (indigo #4F46E5). Single accent only.

## Typography
- Clean geometric / system sans-serif. Large confident headline (clamp ~2.5–4rem). Readable subhead (~1.125–1.25rem).

## Components
- Hero: headline + one supporting sentence + one primary button.
- Button: solid accent fill, rounded (8px), clear hover state.

## Elevation & motion
- Flat and clean. Minimal shadows. Subtle fade-in on load at most; no heavy animation.

## Restraint
No nav bar, no footer link clusters, no extra sections. Exactly one page, one message, one action.
